# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-31

### Added
- Initial release
- Package type selection (Warrant Officer, Recruiting Duty)
- Dynamic document checklists based on program requirements
- Document status tracking (complete, incomplete, N/A, waiver required)
- Progress indicator with percentage complete
- Cover sheet generator with enclosure list
- Routing/endorsement tracker
- PDF export for checklists and cover sheets
- Print functionality
- PWA support for offline use
- LocalStorage for saving package progress
- Dark mode and night (tactical) mode themes
- Mobile-responsive design

### Package Types
- Warrant Officer (Regular/Reserve) - Full checklist from MCRC sample application
- Recruiting Duty - B-Billet SDA screening package
