## Overview

A checklist-driven package builder that helps Marines prepare complete screening packages for special duty assignments and board packages. Track required documents, get step-by-step guidance, generate cover sheets, and ensure package completeness.

**Works offline** · **No account required** · **Data stays on your device**

---

## Features

| Feature | Description |
|---------|-------------|
| **Step-by-Step Guidance** | Phase-based workflow guides you through the entire package process |
| **Official Form Links** | Direct links to download DD forms, NAVMC forms, and access systems (MOL, MCTFS, DISS) |
| **Document Tracking** | Track status (complete, incomplete, N/A, waiver needed) for each document |
| **Cover Sheet Generator** | Generate and print cover sheets with enclosure lists |
| **Routing Tracker** | Track endorsement signatures through the chain of command |
| **Progress Dashboard** | Visual progress bar and stats showing package completion |
| **Live Preview** | Real-time preview of your package as you build it |
| **PDF Export** | Export checklists and cover sheets to PDF |
| **Offline Support** | Works offline as a Progressive Web App (PWA) |
| **Local Storage** | Save and resume package progress anytime |

## Quick Start

1. **Select Package Type** — Choose Warrant Officer, Recruiting, or other package type
2. **Enter Applicant Info** — Provide name, rank, and submission deadline
3. **Follow Step-by-Step Phases** — Work through each phase with detailed guidance
4. **Track Documents** — Mark documents as complete, download forms, add notes
5. **Generate Cover Sheet** — Auto-populated cover sheet with enclosure list
6. **Track Routing** — Monitor endorsement progress through the chain
7. **Export & Submit** — Generate PDFs for your package

## Installation

### Use Online (Recommended)
Visit **[jeranaias.github.io/screening-package-builder](https://jeranaias.github.io/screening-package-builder/)**

### Install as App
1. Open the site in Chrome, Edge, or Safari
2. Click "Install" in the browser menu or address bar
3. Use offline anytime

### Run Locally
```bash
git clone https://github.com/jeranaias/screening-package-builder.git
cd screening-package-builder
# Open index.html in browser, or serve with:
npx serve .
```

## Technology

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Offline**: Progressive Web App (PWA) with Service Worker
- **Storage**: LocalStorage for data persistence
- **PDF**: jsPDF for document generation
- **No Backend**: Runs entirely in your browser

---

## Privacy & Security

- All data stored locally in your browser
- No server communication — works completely offline
- No accounts, no tracking, no analytics
- Your screening package data never leaves your device
- Clear your browser data to remove all stored information
